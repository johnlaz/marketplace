# MarketMaster AI — v1.2

AI-powered single-page PWA for creating and tracking marketplace listings.

## Deploy to GitHub Pages

1. Create a new GitHub repo
2. Upload all files in this folder to the repo root
3. **Settings → Pages → Source: Deploy from branch → main → / (root) → Save**

Live at: `https://yourusername.github.io/reponame/`

## Files
| File | Purpose |
|------|---------|
| `index.html` | Full app — all CSS + JS inline |
| `manifest.json` | PWA install manifest |
| `sw.js` | Service worker (offline cache) |
| `icon192.jpg` | App icon 192×192 |
| `icon512.jpg` | App icon 512×512 |

## Install as App
- **iPhone/iPad:** Safari → Share → Add to Home Screen
- **Android:** Chrome → ⋮ → Install App
- **Desktop:** Address bar install icon in Chrome/Edge

## API Keys
Enter in Settings (⚙) on first launch. Stored locally only.
