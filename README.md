# MarketMaster AI — GitHub Pages PWA

## Deploy in 3 steps

1. Create a new GitHub repo (e.g. `marketmaster`)
2. Upload all files in this folder to the repo root
3. Go to **Settings → Pages → Source: main branch / root** → Save

Your app will be live at: `https://yourusername.github.io/marketmaster/`

## Files
- `index.html` — The full app (all CSS + JS inline)
- `manifest.json` — PWA install manifest
- `sw.js` — Service worker (offline support)
- `icon192.jpg` — App icon 192×192
- `icon512.jpg` — App icon 512×512

## Install as App
- **iPhone/iPad:** Safari → Share → Add to Home Screen
- **Android:** Chrome → ⋮ menu → Add to Home Screen / Install App
- **Desktop Chrome/Edge:** Address bar install icon → Install

## API Keys
Enter your keys in the app Settings (⚙) on first launch. They are stored locally in your browser only.
